About F1 Analytics Platform

A comprehensive data science project combining EDA and machine learning for Formula 1 racing

Project Motivation

Formula 1 is a data-rich sport where milliseconds matter and every decision is backed by analytics. This project was built to demonstrate end-to-end data science capabilities by analyzing over 70 years of F1 history, extracting meaningful insights, and building predictive models that can forecast race outcomes.

The platform serves as both a learning tool for understanding F1 dynamics and a portfolio piece showcasing skills in data engineering, exploratory analysis, machine learning, and full-stack web development.

Data Sources
Primary Data Source

Ergast Developer API provides comprehensive F1 data including race results, driver standings, constructor championships, qualifying times, pit stops, and circuit information from 1950 to present. The API is free, well-documented, and regularly updated.

Machine Learning Approach
Model Architecture

The prediction system uses an ensemble of gradient boosting models (XGBoost, LightGBM, CatBoost) combined with neural networks for multi-class classification of race positions.

Feature Engineering

Over 42 features are engineered including:

  • Driver Features: Recent form, career statistics, circuit-specific performance
  • Constructor Features: Team performance trends, reliability metrics, technical development
  • Circuit Features: Track characteristics, weather patterns, overtaking difficulty
  • Temporal Features: Season progression, championship standings pressure

Model Limitations
  • Cannot predict mechanical failures or reliability issues
  • First-lap incidents and crashes are not modeled
  • Strategy calls (tire choices, pit timing) are simplified
  • Safety car deployments impact actual vs predicted results
